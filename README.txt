# DCQE_simulation
Késleltetett választásos kvantum radír szimuláció

Visual Studio 2019-ben készült és .Net 4-et igényel. 
(Bár aki akarja átteheti a programot régebbi .Net verzióra is, mivel semmit se használ a program, csak a legalapvetőbb .Net komponenseket és funkciókat.)

A három png kép fájl kell a programhoz. be kell másolni az exe fájl mellé.

A program freeware és csak ismeretterjesztési céllal készült.
